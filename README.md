## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/adisreyaj/compito.git
   ```
1. Install NPM packages
   ```sh
   npm install
   ```
1. Configure the environment variables

1. Run the UI
   ```sh
   npm start
   ```
1. Run the API server
   ```sh
   npm start api
   ```
